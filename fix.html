Remnawave SelfSteal Fix

Исправляет Xray Config Profile: Reality → локальный Caddy socket, serverNames → ваш selfsteal-домен. Всё только в браузере, ничего никуда не отправляется.

Что изменено

    Инструкция

    1. Вставь Config Profile из панели Remnawave и укажи selfsteal-домен ноды.
    2. Нажми Исправить конфиг → скопируй JSON → вставь в панель → Save.
    3. У ноды включи все Active Inbounds заново (после смены тегов иначе профиль сбросится).
    4. На сервере ноды выполни одну команду ниже (HY2 certs + проверка).
    5. Проверь: curl -sI https://ДОМЕН/ → HTTP/2 200, Server: Caddy.

    Команда для сервера ноды (одной строкой)

    Укажи домен и нажми «Исправить конфиг»

    Копирует certs в /dev/shm (не symlink — remnanode их не видит через LE), сохраняет копию в /opt/remnanode/certs, ставит @reboot restore, опционально делает /etc/letsencrypt/live/… для чекера Multi-Protocol.
